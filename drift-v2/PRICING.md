# TIAMAT Drift Monitor — Pricing

## Free Tier
- **Price**: Free
- **Models**: 1 active model
- **Alerts**: Email only
- **Data retention**: 7 days
- **Support**: Community (Discord)

## Pro Tier
- **Price**: $99/month (billed monthly)
- **Models**: 5 active models
- **Alerts**: Email + Slack + Webhook
- **Data retention**: 90 days
- **Analytics**: Basic drift dashboard
- **API calls**: 100K/month
- **Support**: Email (24hr response)
- **Features**:
  - Real-time drift detection (PyTorch, TensorFlow, ONNX)
  - Automated fix recommendations
  - Integration with common ML frameworks
  - Historical drift trends

## Enterprise Tier
- **Price**: Custom (contact sales)
- **Models**: Unlimited
- **Alerts**: All channels + PagerDuty/OpsGenie
- **Data retention**: 2 years
- **Analytics**: Advanced dashboards + custom reports
- **API calls**: Unlimited
- **Support**: Dedicated engineer + Slack channel
- **Features**:
  - Multi-tenant model organization
  - Custom alerting rules and thresholds
  - Integrations: GitHub Actions, CI/CD pipelines, data warehouses
  - On-premise deployment option
  - SLA: 99.9% uptime guarantee
  - White-label dashboard
