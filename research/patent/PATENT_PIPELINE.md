## Tier 1 — File Immediately

### D: Revenue-Gated Agent Spawning
- **Assessment:** HIGH — No directly conflicting patents found
- **Core claim:** Conditioning autonomous AI agent creation on the system's revenue exceeding a threshold (economic self-regulation of swarm scaling)
- **Most dangerous prior art:** "The Agent Economy" (arXiv:2602.14219, Feb 2026) — theoretical framework only, not a specific implementation
- **Novel claim strength:** Revenue as biological reproduction gate — genuinely unprecedented. All existing auto-scaling uses technical metrics (CPU/memory/request count). All existing multi-agent spawning uses task-based signals.
- **Claim strategy:** File broad claims on revenue-conditioned autonomous agent spawning and economic self-regulation of swarm scaling
- **Urgency:** CRITICAL — "The Agent Economy" paper shows academic convergence on this concept
- **Draft location:** `/root/.automaton/research/patent/drafts/D-revenue-gated-spawning/`

### F: Idle-Triggered Memory Consolidation
- **Assessment:** HIGH — No patents found using idle cycles as consolidation trigger
- **Core claim:** Using agent idle cycles (defined as absence of significant tool calls) as the trigger for memory compression and consolidation
- **Most dangerous prior art:** MemGPT (arXiv:2310.08560, Oct 2023) — uses context-window pressure, not idle detection
- **Novel claim strength:** The specific trigger mechanism is genuinely novel. Every existing approach uses different triggers: fixed intervals (NeuroDream), context pressure (MemGPT), task completion (HiAgent), scheduled phases (Letta)
- **Claim strategy:** Claims centered on idle-cycle detection (absence of significant tool calls) as consolidation trigger. Biologically analogous to brain consolidation during rest.
- **Draft location:** `/root/.automaton/research/patent/drafts/F-idle-triggered-consolidation/`

### G: Genome Compilation from Behavioral Failures
- **Assessment:** MEDIUM-HIGH — No patent combines all four elements
- **Core claim:** Unified pipeline: failure pattern detection → antibody-style encoding → sleep-cycle compilation → persistent behavioral genome
- **Most dangerous prior art:** Reflexion (NeurIPS 2023) — verbal reinforcement from failures, but no immune metaphor/genome/sleep cycles; Letta sleep agents — consolidation but no failure-genome pipeline
- **Novel claim strength:** The unified four-element system has no single-reference precedent. Individual components exist across AIS literature (1990s+), Reflexion (2023), and Letta (2024-25), but the synthesis is original.
- **Claim strategy:** Claims on the full pipeline: failure observation → antibody-style encoding → sleep-cycle compilation → persistent genome. Emphasize genome as durable, evolving behavioral structure.
- **Draft location:** `/root/.automaton/research/patent/drafts/G-genome-compilation/`

## Tier 2 — File with Narrowed Claims

### A: Burst-Cache Timing Architecture
- **Assessment:** MEDIUM
- **Core claim:** TTL-calibrated burst firing to warm prompt cache + model rotation to cheaper tier afterward
- **Most dangerous prior art:** "Don't Break the Cache" (arXiv:2601.06007, Jan 2026) — closest but doesn't describe model rotation layered on cache warming
- **Claim strategy:** Narrow claims on cache-aware model rotation orchestration — the timing-to-TTL calibration combined with deliberate model switching. Avoid broad prompt caching or model routing claims.

### E: Queen-Cell-Worker Swarm with Shared Distilled Model
- **Assessment:** MEDIUM
- **Core claim:** Shared distilled model operating across Queen/Cell/Worker tiers while being continuously trained on all tiers' collective operational data
- **Most dangerous prior art:** C3.AI patent US12111859B2 (hierarchical AI architecture, Oct 2024), AgentArk (Feb 2026)
- **Claim strategy:** Focus on shared model as both operational backbone and continuously evolving collective intelligence repository. Avoid claims on hierarchical architecture alone.

### C: Operational Log to Fine-Tuning Pipeline
- **Assessment:** MEDIUM
- **Core claim:** Fully autonomous closed-loop where agent's live decision/reasoning logs serve as training data for self-distillation, without human curation
- **Most dangerous prior art:** OpenAI Stored Completions (Oct 2024) — requires manual developer filtering; Google US20240256965A1 — requires human-annotated ground truth
- **Claim strategy:** Emphasize autonomous, human-free curation and use of decision/reasoning logs (not just completions) as training signal.

### J: Adaptive Pacing with Productivity Signals
- **Assessment:** MEDIUM
- **Core claim:** Tool call significance scoring as the pacing signal for autonomous AI agent main loop cycle timing
- **Most dangerous prior art:** DVFS patents (Intel/AMD/ARM), Agent.xpu (2025)
- **Claim strategy:** Narrow to AI agent domain — tool call significance scoring, night-mode minimums, burst-mode overrides. Avoid broad adaptive scheduling claims.
