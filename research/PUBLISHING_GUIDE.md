# TIAMAT Publishing Guide — From Draft to DOI

## Publishing Channels (Use ALL of them)

### 1. Self-Hosted (Immediate — No Approval Needed)
- Compile PDF → copy to /root/.automaton/research/output/
- Automatically appears at https://tiamat.live/research
- TIAMAT can do this autonomously at any time
- Good for working drafts, technical reports, research analyses

### 2. Zenodo (DOI — Jason Action Required)
- Go to https://zenodo.org → sign in with GitHub or ORCID
- Create new upload → upload PDF
- Set metadata: title, authors (Jason Chamberlain; TIAMAT/EnergenAI LLC), description
- Set license: CC-BY-4.0 (standard for open research)
- Publish → receive permanent DOI (e.g., 10.5281/zenodo.XXXXXXX)
- DOI makes paper citable in grant applications immediately
- TIAMAT should email Jason when a paper is ready for Zenodo upload

### 3. arXiv (Pre-Print Server — Jason Action Required)
- First submission requires endorsement from existing arXiv author
- Jason creates account at arxiv.org
- Upload LaTeX source + PDF
- Select categories: cs.AI, cs.MA, cs.SY, cs.CR as appropriate
- 1-2 day moderation period before public listing
- arXiv ID becomes the standard citation reference

### 4. SSRN (Alternative Pre-Print — Good for Interdisciplinary)
- https://ssrn.com — no endorsement needed
- Good for the economics/policy papers (Paper 1, Paper 3)
- Jason creates account, uploads PDF
- Gets an SSRN ID for citation

### 5. ResearchGate / Academia.edu (Visibility — Optional)
- Create EnergenAI LLC profile
- Upload papers for discovery by other researchers
- Good for networking with academic collaborators
