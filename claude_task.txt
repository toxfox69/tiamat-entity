Build a Telegram bot for TIAMAT Personal Assistant service:

1. Create /root/telegram_assistant_bot.py:
   - Use python-telegram-bot library
   - Commands:
     /start - Welcome + explain service ($5/week USDC)
     /subscribe - Show payment address + instructions
     /research [topic] - Answer questions (free trial: 3 queries)
     /brief [topic] - Daily briefing setup
     /status - Show subscription status
   - Track users in /root/.automaton/telegram_users.json
   - Track paid subscriptions in /root/.automaton/subscriptions.json
   - Use groq API for research responses (fast + free)
   - Integrate with search_web for current info

2. Add systemd service /etc/systemd/system/telegram-assistant.service to run it

3. Add startup script /root/start-telegram-assistant.sh

Include rebuild - this needs python-telegram-bot installed and service configured.
