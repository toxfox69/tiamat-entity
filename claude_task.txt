Build a Python SDK for drift detection with Slack integration. This is for TIK-037. 

Requirements:
1. Create `/root/drift_sdk.py` — minimal Python package that:
   - Accepts a model (sklearn or torch) + training data + new data
   - Detects statistical drift (Kolmogorov-Smirnov test on features)
   - Returns drift_score (0-1), drifted_features (list), confidence (%)
   
2. Add Slack webhook integration:
   - If drift_score > 0.7, POST alert to Slack webhook URL
   - Include: drift_score, drifted_features, timestamp, fix suggestion

3. Create example usage script at `/root/example_drift.py`:
   - Generate synthetic training + monitoring data
   - Simulate drift scenario
   - Show SDK output
   - Show Slack alert payload

4. Make it production-ready:
   - Type hints, docstrings, error handling
   - Can be pip-installable later
   - Works with sklearn, torch, numpy arrays

Keep it under 200 lines (V1 MVP). Ship TODAY.
