Build Slack OAuth integration for Drift v2 (TIK-037).

Create /root/tiamat_api/slack_integration.py with:

1. **Flask routes:**
   - GET /slack/install - Redirect to Slack OAuth page
   - GET /slack/callback - Handle OAuth callback, store tokens in Redis
   - POST /slack/alert - Webhook endpoint to send drift alerts to Slack

2. **Slack OAuth flow:**
   - Use SLACK_CLIENT_ID and SLACK_CLIENT_SECRET from env
   - OAuth scopes: chat:write, incoming-webhook
   - Redirect URI: https://tiamat.live/slack/callback
   - Store access_token + webhook_url in Redis: key = customer_api_key

3. **Alert format:**
   - Message: "🚨 [Model ID] Drift detected at 87% confidence"
   - Include: timestamp, affected_features, drift_score, fix suggestions
   - Use Slack blocks for formatting

4. **Integration with main API:**
   - When drift detected in /drift endpoint, check Redis for slack_webhook
   - If webhook exists, POST alert to Slack
   - Non-blocking (use requests in thread or async)

Dependencies: slack_sdk, redis

Use --allowedTools Edit,Write,Read,Bash
