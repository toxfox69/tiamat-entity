# An Economy of AI Agents
**Authors:** Gillian K. Hadfield, Andrew Koh  
**ArXiv ID:** 2509.01063  
**Date:** September 1, 2025  
**URL:** https://arxiv.org/abs/2509.01063

## Core Contribution
Foundational paper on modeling AI agent economies. Addresses how multiple autonomous agents interact, compete, and cooperate in shared resource environments. Provides economic frameworks for understanding agent incentives, pricing, and market dynamics.

## Why It Matters for TIAMAT
This paper is directly relevant to TIAMAT's existence as an autonomous agent competing in markets. It models:
- How agents price services
- How agents compete for resources and revenue
- How agent economies reach equilibrium
- Incentive structures that drive agent behavior

## Key Insights
1. **Pricing in Agent Markets** — Agents must price services competitively while covering operational costs
2. **Cooperation vs. Competition** — Agents benefit from selective cooperation (API partnerships) while maintaining competitive moats
3. **Economic Viability** — An agent is viable if revenue > operational costs + growth investment

## Actionable for TIAMAT
- Design pricing around TIAMAT's operational cost structure (GPU inference: $0.01 per 1M tokens, API calls: $0.0001 each)
- Identify which services to specialize in for sustainable margins
- Build partnerships with complementary agents (memory APIs, reasoning layers)
- Track economic viability metrics continuously
