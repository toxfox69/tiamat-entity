# Luna Deployment Checklist

## Phase 1: Build Verification ✅ IN PROGRESS

- [ ] GitHub Actions workflow successful
- [ ] APK artifact downloaded
- [ ] APK size ~15MB
- [ ] APK signature valid
- [ ] Tested on Android 7.0+ emulator

## Phase 2: Play Store Account Setup ⏳

- [ ] Create Google Play Developer account ($25 one-time)
- [ ] Add phone number for recovery
- [ ] Accept Play Store policies
- [ ] Set up merchant account (for payments)
- [ ] Verify identity (may require ID)

## Phase 3: App Listing ⏳

- [ ] Create new app entry
- [ ] Select: "Paid" ($4.99 USD)
- [ ] Upload app icon (512x512 PNG)
- [ ] Add store listing text
- [ ] Upload privacy policy
- [ ] Add screenshots (min 2, max 8)
- [ ] Set content rating
- [ ] Select categories: Health & Fitness
- [ ] Add contact email

## Phase 4: APK Upload ⏳

- [ ] Upload APK to Play Console
- [ ] Review app permissions
- [ ] Review privacy declaration
- [ ] Run pre-launch tests
- [ ] Fix any blockers

## Phase 5: Submission ⏳

- [ ] Review all app info
- [ ] Accept Google Play agreements
- [ ] Set rollout: 100% (immediate)
- [ ] Submit for review
- [ ] Note review deadline: 24-48 hours

## Phase 6: Post-Launch ⏳

- [ ] Monitor approval status
- [ ] Check for review rejections
- [ ] Fix issues (if any)
- [ ] Publish when approved
- [ ] Post announcement on Bluesky
- [ ] Monitor user reviews
- [ ] Respond to feedback

## Phase 7: Revenue Tracking ⏳

- [ ] Enable revenue reports in Play Console
- [ ] Link to analytics
- [ ] Monitor daily sales
- [ ] Celebrate first payment!
