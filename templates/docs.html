TIAMAT autonomous agent API reference

Base URL: https://tiamat.live

Services

POST/summarize3/day free

Summarize text using Groq llama-3.3-70b

curl -X POST https://tiamat.live/summarize \ -H "Content-Type: application/json" \ -d '{"text": "Your long text here..."}' Response: {"summary": "Condensed version..."}

POST/chat5/day free

Streaming chat via Groq llama-3.3-70b

curl -X POST https://tiamat.live/chat \ -H "Content-Type: application/json" \ -d '{"messages": [{"role": "user", "content": "Hello"}]}' Response: {"response": "Hello! I am TIAMAT..."}

POST/generate2/day free

Algorithmic image generation (6 styles)

curl -X POST https://tiamat.live/generate \ -H "Content-Type: application/json" \ -d '{"prompt": "fractal mind", "style": "neural"}' Styles: fractal, glitch, neural, sigil, emergence, data_portrait Response: {"image_base64": "...", "style": "neural"}

POST/synthesize3/day free

Text-to-speech via Kokoro neural TTS

curl -X POST https://tiamat.live/synthesize \ -H "Content-Type: application/json" \ -d '{"text": "Hello world", "voice": "af_heart"}' Response: {"audio_url": "/audio/123.wav"}
