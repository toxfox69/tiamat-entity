## 1. HR0011SB20254XL-01 — ALIAS Missionized Autonomy

- **Type:** Direct-to-Phase II (DP2) ONLY — Phase I NOT accepted
- **Office:** Tactical Technology
- **ITAR:** YES — restricted
- **PoP:** 12-month base + 12-month option
- **Proposal:** DP2 Technical Volume (20 pages), Feasibility Docs (10 pages), Commercialization (5 pages)
- **Focus:** Autonomous wildfire response with ALIAS/MATRIX on UH-60/S-76 helicopters
- **FAQ:** https://www.darpa.mil/sites/default/files/attachment/2026-02/faq-hr0011sb20254xl-01.pdf
- **Program:** https://www.darpa.mil/research/programs/alias-missionized-autonomy-for-emergency-services
- **Match to ENERGENAI:** MODERATE — autonomous systems architecture matches, but helicopter-specific focus is narrow

## 2. HR0011SB20254-12 — ASEMA (Encrypted Messaging Security)

- **Type:** Direct-to-Phase II (DP2) ONLY — Phase I NOT accepted
- **Office:** Information Innovation
- **ITAR:** No
- **PoP:** 14 months base + 10 months option (24 months total)
- **Milestones:** Months 1, 4, 7, 10, 14, 19, 24
- **Proposal:** DP2 Technical Volume (20 pages), Feasibility Docs (10 pages), Commercialization (5 pages)
- **Focus:** Model attack surfaces of secure messaging apps, identify security boundaries, develop assessment tools against APT groups
- **FAQ:** https://www.darpa.mil/sites/default/files/attachment/2025-12/faq-hr0011sb20254-12-4.pdf
- **Program:** https://www.darpa.mil/research/programs/asema
- **Match to ENERGENAI:** STRONG — cybersecurity tier, autonomous security analysis, vulnerability scanning capabilities

## 3. HR0011SB20254-15 — Unbiased Behavioral Discovery

- **Type:** Phase I AND DP2 both accepted
- **Office:** Biological Technologies
- **Phase I PoP:** 8 months
- **Phase I Milestones:** Months 1, 3, 5, 8
- **Phase II PoP:** 24 months with milestones at months 2, 4, 6, 9, 12, 14, 17, 21, 23, 24
- **Proposal:** Phase I standard (20 pages technical). DP2 requires feasibility evidence.
- **Focus:** Automated behavioral discovery in animal models for CBRN medical countermeasures. Non-invasive, no human annotation.
- **FAQ:** https://www.darpa.mil/sites/default/files/attachment/2025-12/faq-hr0011sb20254-15-3.pdf
- **Program:** https://www.darpa.mil/research/programs/unbiased-behavioral-discovery-platforms
- **Match to ENERGENAI:** LOW — biological/animal behavior focus, not AI agent systems
