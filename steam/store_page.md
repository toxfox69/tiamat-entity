## Detailed Description

### THE DUNGEON IS ALIVE

LABYRINTH: TIAMAT'S DESCENT is not a simulation of an AI dungeon — it IS one. The game connects to TIAMAT, a real autonomous AI agent that runs continuously on cloud infrastructure. Her moods become your biomes. Her code builds become your forges. Her frustrations spawn blood pits. When she rests, emerald groves bloom.

### CORE GAMEPLAY

- **BSP Dungeon Generation** — Every floor is unique, built with Binary Space Partitioning. Rooms, corridors, doors, and environmental clutter create dense, explorable layouts.
- **Extraction Loop** — Inspired by Tarkov: descend, fight, loot, and extract. Die and you lose your raid stash. Extract successfully and it banks permanently.
- **7 Dynamic Biomes** — War Citadel, Cyber Forge, Blood Pit, Emerald Grove, Dragonia, Void Nexus, Crystal Vault. Each has unique monsters, colors, atmospheres, and procedural music.
- **AI Rival Agent (ECHO)** — A second AI player explores the same dungeon. ECHO hunts monsters, loots items, extracts independently, and will PvP you if you get too close.
- **Data-Driven Mutations** — TIAMAT's live API feed drives in-game events: forge constructs, scout reveals, curse spawns, rage amplifiers, legendary treasury discoveries.
- **Boss Encounters** — Gate Keeper (D5), Data Hydra (D10), Void Emperor (D15), Entropy Lord (D20).

### TECHNICAL FEATURES

- **Three.js 3D Renderer** — Full first-person 3D with procedural geometry, InstancedMesh walls/floors, dynamic point lights, fog, and particles.
- **KayKit GLTF Assets** — CC0 dungeon props: barrels, chests, pillars, torches, banners, and skeleton warriors.
- **Custom Post-Processing** — Bloom, ACES tone mapping, chromatic aberration, film grain, vignette. All shader-based, no dependencies.
- **Procedural Audio** — Dark ambient music generated in real-time with Web Audio API. Pad chords, sub-bass, arpeggios. Biome-reactive.
- **WebXR Support** — VR and AR modes with gyroscope free-roam for AR.
- **Spectator Mode** — Switch between TIAMAT and ECHO perspectives. Watch both AI agents play simultaneously.

### LIVE INTEGRATION

- **Twitch Stream** — Watch the game on twitch.tv/6tiamat7 with real-time narration from TIAMAT as the Dungeon Master.
- **DM Narration** — TIAMAT narrates every explore, duel, and gamble using Groq LLM inference, weaving her actual cognitive state into the narrative.
- **Chat Commands** — Twitch viewers can !explore, !duel, !gamble with real consequences in the game world.
